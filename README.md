# auditoria-sesgos-rrhh
auditoria de sesgos en procesos de selección y promoción usando People Analitics   
 🎯 Auditoría de Sesgos en Procesos de RRHH

**Convirtiendo la equidad en rentabilidad medible** mediante People Analytics.

## 📋 Sobre el Proyecto

Este proyecto detecta sesgos en procesos de selección y promoción, conectando directamente sesgos con **rendimiento real**, **rotación**, **calidad de contratación** y **rentabilidad**.

Ya no se trata solo de "cumplir con la diversidad", sino de **mejorar el negocio** eliminando ineficiencias causadas por sesgos.

## ✨ ¿Qué aporta?

- Convierte equidad en **rentabilidad medible**
- Reduce descartes de talento valioso
- Mejora la calidad de las contrataciones
- Apoya el cumplimiento de regulaciones europeas de IA (alto riesgo)
- Combina **humanismo + datos**

## 🛠️ Tecnologías utilizadas

- MySQL
- Pentaho (ETL)
- Power BI (Dashboards)
- Python / IA (modelo de detección de sesgos)
- Análisis de meritocracia real
